二分搜索

  • 二分搜索： 对于有序的序列进行二分查找。
    　　代码如下；

      bool find(int num, int a[], int st, int end)
  {
      if(st > end)
          return false;

      if(num == a[(st + end) / 2])
          return true;

      if(num < a[(st + end) / 2])
          return find(num, a, st, (st+end)/2 - 1);

      if(num > a[(st + end) / 2])
          return find(num, a, (st+end)/2 + 1, end);
  }
