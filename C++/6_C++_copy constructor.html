拷贝构造函数

  • 作用：
    　　很多对象初始化是用过相同类型的对象赋值进行初始化的，当然也有传参 以及函数返回。

      class lan
  {
      private:
      int age;

      public:
      lan(){ age = 18;}
      lan(const lan& f){ age = f.age; }
      ~lan(){ cout<<"over"<<endl; }
  }

　　对于类中具有动态成员的情况，用直接对应的成员来赋值是不可以的。这就用到了拷贝 构造函数，来“深拷贝”。 

        class lan
        {
            public:
            int age;
            static int num;
            int *money;
            lan()
            {
                age = 18;
                num ++;
                money = new int(10);
            }
            lan(const lan& f)
            {
                age = f.age;
                num++;
                money = new int(10);
                for(int i=0;i<10;i++)
                    money[i] = f.money[i];
            }
        };
        int lan::num =0;

　　类中的int *money就是动态类型。如果浅拷贝，那么两个对象a、b的money指针会 指向同一内存区域，显然，这是不合适的。因此需要在拷贝构造函数里重新分配内存。
