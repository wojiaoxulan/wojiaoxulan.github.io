快排算法

  • 说明：
    　　分而治之，代码如下：

      void quicksort(int *a, int l, int r)
  {
      if(l >= r)
          return;

      int i = l, j = r, tem = a[l];
      while(i < j)
      {
          while(a[j] >= tem)
              j--;
          if(i<j)
              a[i++] = a[j];

          while(a[i] < tem)
              i++;
          if(i<j)
              a[j--] = a[i];
      }
      a[i] = tem;
      quicksort(a, l, i-1);
      quicksort(a, i+1, r);
  }

    　　唉。。。一般直接调用函数了，谁没事写快排算法啊。烦人。。。
