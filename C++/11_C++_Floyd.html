Floyd算法

算法思想：用三重循环不断进行更新邻接矩阵，其中邻接矩阵D[i][j]代表顶点i到顶点j的几何距离。 

    a)　初始化：D[u,v]=A[u,v]
    b)　For k:=1 to n
            For i:=1 to n
                For j:=1 to n
                    If D[i,j]>D[i,k]+D[k,j] Then
                        D[i,j]:=D[i,k]+D[k,j];
    c)　算法结束：D即为所有点对的最短路径矩阵
