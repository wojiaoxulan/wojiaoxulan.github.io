多态

  • 多态的定义：
    　　多态即接口不与某一函数进行绑定，也就是不会在编译时进行绑定， 而是在运行的时候确定使用哪种实现方式，这也是C++的三大特性之一。
    　　示例如下：

      #include "stdafx.h"
  #include "iostream"
  using namespace std;
  class baba
  {
      public:
          int val;
          virtual void print() { cout<<"偶都桑"<<endl; }
          baba()               { val = 0; }
          baba(int num)        { val = num; }
  };
  class era:public baba
  {
      public:
          void print() { cout<<"鼓捣猫"<<endl; }
          era(int num) { val = num; }
          era()        { val = 0; }
  };
  int main(int argc, char* argv[])
  {
      baba *parents;

      parents = new era(-1);
      parents->print();

      parents = new baba(2);
      parents->print();

      return 0;
  }

　　输出的结果如下： 

    鼓捣猫
    偶都桑
  • 实现的方式：
    　　其实就是对于每一个类构造一个虚表，虚表里存放了虚函数的地址。 而每一个对象都会有一个虚表指针，注意：虚表指针是在对象里的。 虚表指针指向该对象所属类的虚表。
    　　因此当使用父类指针指向子类对象时，会根据对象的类型，将虚表指针进行初始化，这样就可以实现多态了。
