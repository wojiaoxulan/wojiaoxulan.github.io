二叉查找树

  • 数据结构：
    　　这个数据结构哇，还是很简单滴，之前都是用结构体， 现在用类来实现，也是一样滴。构造函数还是要做好滴。
      class node
  {
      public:
          node * left;
          node * right;
          int val;
          node(){
              left  = NULL;
              right = NULL;
              val  = 0;
          }
          node(int a){
              left  = NULL;
              right = NULL;
              val  = a;
          }
  };

  • 插入节点:
    　　哎呀，就是构造树。我是用递归做的，虽然可以用非递归，但是这样代码看起来 很简洁。不过还是不要这么用。

      /// @brief 插入节点
  /// @param[in]  head 头结点
  /// @param[in] num  要插入的数字
  void insert(node *&head, int num)
  {

      if(head == NULL)
      {
          node *N = new node(num);
          head = N;
          return;
      }

      if(num <= head->val)
      {
          if(head->left == NULL)
          {
              node *N = new node(num);
              head->left = N;
              return;
          }
          insert(head->left, num);
      }
      else
      {
          if(head->right == NULL)
          {
              node *N = new node(num);
              head->right = N;
              return;
          }
          insert(head->right, num);
      }
  }


  • 删除节点:
    　　有插入就有删除。删除比较麻烦，因为鬼知道会删除哪一个节点， 而该节点删除后很有可能破坏二叉查找树的结构。删除节点主要有三种情况：
    　　1. 待删除节点是叶子节点。此时释放掉节点空间，然后指针置为NULL；
    　　2. 待删除节点的做字数或者右子树为空。此时用右子树或者左子树代替待删除节点；
    　　3. 待删除节点的左右子树均不为空。此时分为两种情况：
    　　　　（1）如果右子树的左子树为空，则 将待删除点的左子树变成待删除点右子树的左子树，然后待删除点的右子树取代待删除点。 对称的情况是一样的；
    　　　　（2）找到待删除点的后继，然后取代待删除点。

      void del(node *&head, int num)
  {
      node *p = head;
      if(head == NULL)
      {
          cout<<"木有找到"<<endl;
          return;
      }
      if((head)->val  == num)
      {
          if(p->left == NULL && p->right ==NULL)
          {
              head = NULL;
              delete p;
          }
          else if(p->left == NULL)
          {
              p = head->right;
              *head = *(head->right);
              delete p;
          }
          else if(p->right == NULL)
          {
              p = head->left;
              *head = *(head->left);
              delete p;
          }
          else 
          {
              if(p->right->left == NULL)
              {
                  p = head->right;
                  head->right->left = head->left;
                  *head = *(head->right);
                  delete p;
              }
              else
              {
                  node *s = head->right;
                  while(s->left != NULL)
                      s = s->left;
                  *head = *s;
                  delete s;
              }
          }
          return;
      }
      if(head->val  > num && head->left !=NULL)
          del(head->left, num);

      else if(head->val  < num && head->right !=NULL)
          del(head->right, num);
  }

  • 查找节点:
    　　很简单，不说了。上代码！
      node *find(node *head, int num)
  {
      if(head == NULL)
          return NULL;
      if(num == head->val)
          return head;
      else if(num < head->val)
          find(head->left, num);
      else
          find(head->right, num);
  }
