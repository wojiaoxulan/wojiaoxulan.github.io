变量和基本类型

1. extern: 关键字，放在变量或者函数之前，表示声明。但是不能对变量进行初始化，否则extern将失效。

2. ：： : 放在变量之前，表示访问全局变量。

3. 引用不是对象，而是和对象进行绑定。一般来说，这种绑定是不可变的。

4. const int 变量： 常量引用，限制了引用对于绑定对象的操作可以绑定非常量，但是不能通过引用改变对象的值。

5. 只能用指向常量的指针来指向常量，同时指向常量的指针也可以指向非常量。

6.常量指针： 

int *const 变量

指针指向的地址是只读的。

7. 当变量的类型比较复杂时，从右往左进行分析。靠近变量名的修饰符对变量起直接作用。
